# Synonymous

A lightweight browser extension to quickly find the meaning of words directly on the page you are reading.

## Features

*   **Seamless Integration**: Select any word or short phrase (up to 5 words), and a floating "📖 Define" button will appear right next to it.
*   **Quick Definitions**: Click the button to instantly view the definition and part of speech in a clean popover.
*   **Automatic Theme**: The popover automatically adapts to your system's light or dark theme (features a Catppuccin Mocha dark mode).
*   **Easy Dismissal**: Click anywhere outside the popover or press the `Esc` key to close it.

## Usage

1.  Select a word on a webpage.
2.  Click the **📖 Define** button that pops up below your selection.
3.  Read the definition!
